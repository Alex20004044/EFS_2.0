When you create new project follow this steps:

1. Move this folder to the root of hierarchy
2. Open "PersistentSystems" scene and configure it. Add all necessary systems. They will be initialized when you enter play mode
3. Add "PersistentSystems" scene to bulid index
4. Add all necessary systems in DevelopmentSystems. Don't add it to build index!
5. You can disable scene initializing in "ProjectInitConstants" script
